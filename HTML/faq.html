Important FAQ

What is the Document Object Model?

The Document Object Model (DOM) is an object based model where we can use javascript to manipulate html document such as add, remote, and modify elements. In this model each of html element can be considered as object them we can change object property and attributes using various methods in DOM

How do you select on the element from the DOM?

The easiest way to access a single element in the DOM is by its unique ID. You can get an element by ID with the getElementById() method of the document object. Then can change class, value, and styles of the selected element.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. Thus, we can control the behavior of an event using eventListener.

How do you manipulate on element's attributes and styles using the DOM?

Changing an element's style (CSS) attributes is one of the ways in which the DOM can be used. We can set styles of HTML document or we can directly use html block and append it as a child node.
